40.清除浮动的必要性：

在很多时候，你想要对页面进行布局，你需要先设置一个大的盒子，然后再设置几个小的盒子，然后进行浮动，达到盒子与盒子之间无间距的目的，但是使用浮动的话，当前的盒子所在的位置就会消失，因此，当盒子只有一排的时候，可能会出现当前排都消失的情况。所以需要清除浮动的影响。
